1. Сверстать html форму с полями: 
- name,
- last name,
- email,
- phone

2. С помощью Ajax передать данные формы после сабмита в php файл методом POST
3. В php файле получить данные, сделать валидацию, и вернуть обратно
4. После сабмита формы: 
- в случае успешной валидации вывести только что введенные данные
- в противном случае вывести ошибку валидации (e.g. Invalid email)


Требования: кроме вышеперечисленного - полная свобода. Оцениваться будет только php и js.
Результат можно залить в публичный гит-репозиторий, либо отправить на почту архив
